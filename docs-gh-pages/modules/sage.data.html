sage.data package

SAGE data package.

This package provides functionality for loading, profiling, and analyzing data from various sources.

load_data(file_path, sheet_name=None, **kwargs)[source]

Load data from a file into a pandas DataFrame.

Supports various file formats including CSV, Excel, JSON, and parquet.

Parameters:

  • sheet_name (str | None) – For Excel files, the name of the sheet to load

  • **kwargs – Additional arguments to pass to the pandas read functions
